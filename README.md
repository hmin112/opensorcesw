### **1) top**

+ 시스템의 상태를 전반적으로 가장 빠르게 파악 가능
+ 옶션 없이 입력하면 interval 간격(기본 3초)으로 화면을 갱신하며 정보를 보여줌
+ q를 눌러 작동 정지 가능
+ 실행 상태에서 다양한 명릉을 입력하여 프로세스 상태를 출력하거나 제어가 가능

|옵션|내용|
|---|---|
|-d[second]|지정한 초(second)마다 갱신|
|-p[process id]|지정한 프로세스 ID의 정보만 출력|
|-c|커맨드를 실행 옵션을 포함해서 출력|
